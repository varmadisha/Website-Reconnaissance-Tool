
# 🌐 Website Reconnaissance Tool  

## 🔐 Overview  
This project is a **Website Reconnaissance Tool** developed for cybersecurity learning and ethical hacking purposes.  
It helps in gathering critical information about a target website, which is the first step in vulnerability assessment.  

---

## 🚀 Features  

- 🔍 Domain Information Gathering  
- 🌐 Subdomain Enumeration  
- 🚪 Port Scanning (Common Ports Detection)  
- 🔒 SSL Certificate Analysis  
- 📡 DNS Information Retrieval  
- ⚡ Fast & User-Friendly Interface  

---

## 🛠️ Technologies Used  

- Python  
- Tkinter (GUI)  
- Socket Programming  
- Requests Library  
- DNS Resolver  

---

## 🧠 How It Works  

The tool collects publicly available information about a target website such as:
- IP address  
- Open ports  
- Subdomains  
- SSL details  

This helps security professionals understand the attack surface of a web application.

---

## ⚠️ Disclaimer  

This tool is created for **educational purposes only**.  
Do not use it on unauthorized websites.  
Always take permission before performing any security testing.
